Real-Time Facial Expression Detection (Heuristic, dlib)\
\pard\pardeftab720\sa240\partightenfactor0

\f1\b0\fs24 \cf0 This project is a real-time facial expression detector built with 
\f0\b OpenCV
\f1\b0  and 
\f0\b dlib
\f1\b0 .\uc0\u8232 It uses facial landmarks and simple geometric heuristics (not machine learning) to classify expressions and react to them with spoken comments.\
The goal of this project is to explore how far you can go with 
\f0\b lightweight, interpretable rules
\f1\b0  instead of heavy emotion-classification models, while keeping everything fast and explainable.\
